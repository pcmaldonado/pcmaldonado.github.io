Car Price Estimator

From data collection to deployment

Project Overview

  • Scraped around 50,000 ads of selling cars from a popular french website using Python (BeautifulSoup)
  • Engineered features from additional scraped data on brands (luxury brands & origin)
  • Explored 9 different models, then applied GridsearchCV to the better performing ones to get the best model
  • Built a web application using Flask that lets users input car features to get an estimate of its selling price

Project information

  • Category: Regression
  • Data: Collected around 50,000 ads of selling cars from a popular french ad website. In addition, also scraped data related to car brands to better understand the "Brand" feature
    (luxury & origin)
  • Motivation: for people that do not know much about cars it can be difficult to evaluate the price of a car before selling/buying one
  • Objective: create a web application able to esimate the selling price of a car given a number of features
  • Main Tools: BeautifulSoup, Flask, XGBoost, Sklearn, Matplotlib
  • Deployed on: Heroku
  • Analysis Code: Github
  • Deployment Code: Github
